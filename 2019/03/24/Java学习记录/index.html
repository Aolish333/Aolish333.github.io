阅读总结

  1. 学习先修技术、准备源码；
  2. 查找相关资料了解所以学习源码的结构；
  3. 阅读源码善于用Debug功能；
  4. 写笔记、画图、总结。

第一次如何阅读源码

我的方法：

  1. 了解IDEA的Debug功能；
  2. 阅读自己要看的代码（按照阅读总结里的）；
  3. 查找网上相关的博客与自己的阅读笔记进行对比
  4. 记录欠缺的地方。
