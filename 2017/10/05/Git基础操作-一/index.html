初识Git

git的定义

Git是一个开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。
Git 是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件。
Git 与常用的版本控制工具 CVS, Subversion 等不同，它采用了分布式版本库的方式，不必服务器端软件支持。

git commit 进行提交

使用 git add 命令将想要快照的内容写入缓存区， 而执行 git commit 将缓存区内容添加到仓库中。

git commit -a 跳过git add提交缓存

为此，我们将它们提交到仓库。
当我们修改了很多文件，而不想每一个都add，想commit自动来提交本地修改，我们可以使用-a标识。。命令格式如下：

git commit -a

你会进入git bash 编辑界面，关于如何退出请参考下文。
如果您不使用-m，会出现编辑器来让你写自己的注释信息。

git commit -m "Adding files"

git commit 命令的-a选项可将所有被修改或者已删除的且已经被git管理的文档提交到仓库中。
千万注意，-a不会造成新文件被提交，只能修改。

git rm 将条目从缓冲区中移出

git mv 移动或重命名一个文件、目录、软连接。

发布版本到Github

刚才才发现github上你创建的仓库已经有教程了，所以看下图就可以。

echo "# myFolder" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin https://github.com/Aolish333/myFolder.git
git push -u origin master
