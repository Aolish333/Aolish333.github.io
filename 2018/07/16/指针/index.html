指针的一些疑惑

int &p； 是不正确的，与p靠得最近，所以p是一个指针，但是这个指针的类型是int &,所以这个是错吴的， 因为不能创建引用的指针。

int & p, 是正确的， &与p靠得最近，所以p是一个引用，这个引用的类型是int，也就是一个指针变量的引用，但是必须初始化，否则也是错误的。
